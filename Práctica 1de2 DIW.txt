Diseño de Interfaces Web – 2.º DAW
Proyecto: Showcase de CSS moderno
Objetivo de la práctica
Desarrollar un proyecto web demostrativo cuyo objetivo sea aplicar y mostrar el uso de
CSS moderno, trabajando:
•

Pseudoclases

•

Animaciones CSS

•

Estilos avanzados con imágenes

•

Iconos con Font Awesome

•

Tipografías externas

•

Diseño responsive

Esta práctica no se plantea como un trabajo “para el profesor”, sino como un proyecto
personal y profesional, pensado para que el alumno pueda mostrarlo públicamente como
parte de su portfolio.

2

Enfoque profesional del proyecto (MUY IMPORTANTE)
Este proyecto debe entenderse como:
Una pieza de portfolio personal
Un ejemplo real de lo que sabes hacer con CSS
Un trabajo que puedes enseñar a empresas
Por ese motivo:
•

El diseño debe ser cuidado

•

El código debe estar limpio

•

El resultado debe ser presentable

No se evalúa solo que funcione, sino que pueda enseñarse con orgullo.

Descripción general
El proyecto debe funcionar como un showcase visual de CSS moderno.
No es obligatorio que sea una web corporativa completa.
Puede ser:

3

•

Página creativa

•

Mini landing visual

•

Galería de estilos

•

Proyecto experimental de diseño
